What changes in lib/pdf.ts

Element Currently With branding
Grid borders rgb(0, 0, 0) brand.colorPrimary → rgb()
Title text rgb(0, 0, 0) brand.colorPrimary → rgb()
Logos Static files from /public Fetched from brand storage URLs
QR foreground Black (#000000) brand.colorPrimary
Font Helvetica (standard) Helvetica (unchanged — custom PDF fonts are complex, low value)

💡 PDF fonts stay as Helvetica — embedding custom fonts in pdf-lib requires font file bytes and adds significant complexity for minimal visual gain on a printed bingo card. Web fonts handle the brand personality; PDFs stay clean and functional.

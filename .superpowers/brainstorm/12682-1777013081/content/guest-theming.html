What adapts per brand

Element Source
Background gradient Generated from --brand-primary + --brand-accent (same radial pattern, different colours)
Header bar --brand-primary background, --brand-accent border
Header logo Brand's dark-background logo from storage
Content cards --brand-primary background, --brand-accent border
QR code foreground --brand-primary
Break message brand.breakMessage
End message brand.endMessage
Footer bar --brand-primary-light border, --brand-primary background
Font --brand-font (loaded dynamically via Google Fonts)

💡 The gradient pattern (radial circles + linear base) stays the same structurally — only the colour inputs change. This keeps the "dust effect" aesthetic consistent while adapting to each brand.
